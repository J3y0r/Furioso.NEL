FuriosoNEL
===========

## 本项目基于基于 [Codexus.OpenSDK](https://github.com/corpnetease/Codexus.OpenSDK) 以及 Codexus.Development.SDK 和 [FuriosoNEL](https://github.com/FandMC/FuriosoNEL)

### 安装要求
- .net 9.0

### 安装步骤
1. 下载最新的构建结果
2. 双击打开即可

### 怎么装插件？
*FuriosoNEL* 插件商城已经不可用，后续我会推出新的 Plugin API Docs
